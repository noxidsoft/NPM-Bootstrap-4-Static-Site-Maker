## Download

```
$ git clone https://github.com/noxidsoft/NPM-Bootstrap-4-Static-Site-Maker.git
```

## Prepare packages

```
$ npm install
```

## Bash based (Linux, Git bash, etc)

```
$ npm run dist-bash
```

## Windows

```
> npm run dist-win
```
